A Hierarquia de Informação: Títulos e a Lógica de Relevância

No desenvolvimento moderno, os títulos não servem para definir o tamanho da letra ou deixar o texto em negrito. No HTML, as tags de <h1> a <h6> servem para criar a árvore de relevância do seu conteúdo.

  • Organização em Níveis: Pense nos títulos como o sumário de um livro. O <h1> é o título da obra (deve ser único por página), enquanto o <h2> define os capítulos principais e o <h3> os subtopicos.

  • Navegação e SEO: Essa estrutura é o que permite que ferramentas de busca, como o Google, entendam sobre o que você está escrevendo. Além disso, tecnologias assistivas (leitores de tela) usam os títulos para permitir que usuários saltem entre as seções do seu site.

  • Regra Semântica: Nunca escolha um título baseado na aparência. Escolha-o baseado no lugar que aquela informação ocupa na hierarquia do seu texto.

Fluxo de Conteúdo: Formatação de texto

Se as seções e títulos formam o esqueleto do site, as tags de formatação de texto são os nervos que dão sensibilidade ao conteúdo. No HTML5, formatar um texto não é apenas sobre "estética", mas sobre informar ao navegador e aos motores de busca o valor real de cada palavra ou frase.

1. O Fluxo de Texto e a Quebra de Linha

Diferente de um editor de texto comum, o HTML ignora múltiplos espaços ou ENTER dados diretamente no código-fonte.

  • Seguimento Contínuo: Um parágrafo (<p>) mantém o fluxo do texto constante para se adaptar ao tamanho da tela do usuário.
  • Quebra Manual: Quando precisamos forçar uma descida de linha sem criar um novo parágrafo, utilizamos a tag <br>. Ela é uma tag "vazia" (não possui fechamento) e deve ser usada com moderação para não prejudicar a responsividade.

3. Marcações Especiais e Notações Técnicas

Para conteúdos que fogem do texto comum, o HTML oferece ferramentas de precisão:

  • Texto Marcado (<mark>): Funciona como um "marca-texto" amarelo, ideal para destacar termos de busca ou passagens relevantes.
  • Escrita Científica: Utilizamos <sup> para sobrescrito (ex: x2) e <sub> para subescrito (ex: H2O).
  • Abreviações (<abbr>): Permite que, ao passar o mouse sobre uma sigla, o navegador exiba seu significado completo através do atributo title.

4. Citações e Referências Externas

Trabalhar com textos acadêmicos ou blogs exige dar crédito às fontes. O HTML possui duas formas principais:

  • Citações Curtas (<q>): Usada para frases inseridas dentro de um parágrafo. O navegador geralmente adiciona as aspas automaticamente.
  • Citações Longas (<blockquote>): Cria um bloco isolado para citações extensas. O atributo cite dentro dessa tag é usado para indicar a URL original da fonte, ajudando no SEO e na credibilidade do conteúdo

5. O Universo do Código: <code> e <pre>

Para exibir textos onde a indentação e os caracteres especiais devem ser preservados:

  • <code>: Formata um pequeno trecho de código com fonte monoespaçada (onde todas as letras têm a mesma largura), facilitando a leitura técnica.
  • <pre>: Abreviação de "pré-formatado". Esta tag é única pois ela respeita exatamente os espaços e quebras de linha que você digitar no código, sendo perfeita para exibir blocos inteiros de linguagens de programação.
